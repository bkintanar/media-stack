# Media Stack Setup
1. `mkdir -p ~/media-stack/data/torrents/{movies,tv,music}`
2. `mkdir -p ~/media-stack/data/media/{movies,tv,music}`
3. `mkdir -p media-stack/{qbittorrent,prowlarr,radarr,sonarr,bazarr,jellyfin,recyclarr}/config`
4. `chmod -R 755 ~/media-stack/data`
5. `cd media-stack`
6. docker compose up -d`
